Flutter与Swift对比

变量声明

Swift和Dart都可以自动推断类型
Swift在变量名之后带上类型，已<:>分隔，Dart则在变量名之前需要写明类型

Swift
var name:String
var name:String = “Tom”

Dart
String name;
String name = ‘Tom’;
